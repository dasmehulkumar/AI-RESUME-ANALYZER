<h1 align="center">🚀 AI Resume Analyzer (ATS-Based)</h1>
<h3 align="center">🧠 Smart Resume Screening using NLP & AI</h3>

<p align="center">
  Transform your resume into a job-winning document with AI-powered analysis, ATS scoring, and intelligent suggestions.
</p>

<hr>

<h2>📌 Overview</h2>
<p>
<strong>AI Resume Analyzer</strong> is a full-stack web application that evaluates resumes against job descriptions using advanced Natural Language Processing techniques. It provides a <b>realistic ATS score</b>, highlights skill gaps, and offers actionable suggestions to improve resume quality.
</p>

<p>
Built using modern AI models like Sentence Transformers and BERT, this tool simulates how real companies filter candidates.
</p>

<hr>

<h2>✨ Key Features</h2>
<ul>
  <li>📄 PDF Resume Upload</li>
  <li>🧠 Semantic Similarity Matching (AI-based)</li>
  <li>📊 ATS Score (0–100) with progress bar</li>
  <li>✅ Matched Skills Detection</li>
  <li>❌ Missing Skills Identification</li>
  <li>🛠 Resume Improvement Suggestions</li>
  <li>🎨 Modern UI with Tailwind CSS</li>
  <li>⚡ Fast Backend using Flask</li>
</ul>

<hr>

<h2>🧠 How It Works</h2>

<pre>
Upload Resume → Extract Text → Compare with Job Description 
→ Skill Matching → ATS Score → Suggestions
</pre>
